# Generated Artifacts

Files in this directory are generated by repository scripts and checked in for agent legibility.

- `backend-openapi-summary.json` is generated from `backend_openapi.json`.
- `eslint-baseline.json` is generated from current ESLint output and freezes legacy violations.

Refresh commands:

```bash
bun run generate:api-summary
bun run generate:eslint-baseline
```
