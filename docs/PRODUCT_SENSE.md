# PRODUCT_SENSE

Verve helps users review, organize, and improve outdoor activity data.

## Product Beliefs
- Activity data should be trustworthy and easy to compare.
- Forms should reduce ambiguity around units, types, tags, equipment, and locations.
- Maps, charts, and highlights should clarify the activity instead of decorating it.
- API failures should be handled plainly and recoverably.

## Agent Guidance
- Preserve existing workflows unless the task explicitly changes them.
- Prefer small, understandable improvements over broad redesigns.
- When user-facing behavior changes, update docs and validation checks in the same change.
